# discord.js-toolbox

A collection of helpers I kept rewriting across every bot I built. Instead of copy-pasting the same moderation safety checks, embed boilerplate, and pagination logic into every project, I packaged it up.

Everything here is built for **discord.js v14** and assumes you're writing a real bot — not a tutorial project. The functions are small, focused, and don't try to abstract too much.

## What's in the box

## Requirements

- Node.js 18 or higher
- discord.js v14
